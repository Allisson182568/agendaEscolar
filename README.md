# Política de Privacidade - Agenda Escolar IA

**Última atualização:** 29 de Dezembro de 2025

A sua privacidade é importante para nós. É política do **Agenda Escolar IA** respeitar a sua privacidade em relação a qualquer informação sua que possamos coletar no aplicativo.

## 1. Coleta de Informações
Solicitamos informações pessoais apenas quando realmente precisamos delas para lhe fornecer um serviço. Fazemo-lo por meios justos e legais, com o seu conhecimento e consentimento.

### Dados Coletados:
* **Identificação:** Nome, e-mail e telefone (para login e comunicação).
* **Dados Escolares:** Informações sobre alunos, turmas, notas e frequência.
* **Uso do App:** Logs de interação para melhoria do sistema.

## 2. Uso de Inteligência Artificial (AI)
Nosso aplicativo utiliza serviços de Inteligência Artificial (Google Gemini) para fornecer funcionalidades como:
* **Radar de Evasão:** Análise de padrões para prevenir saída de alunos.
* **Magic Write:** Auxílio na redação de comunicados.
* **Resumos Automáticos:** Geração de relatórios de desempenho.

Os dados processados pela IA são anonimizados sempre que possível e utilizados estritamente para fornecer os insights dentro do aplicativo. Não vendemos dados para treinamento de modelos externos de terceiros.

## 3. Armazenamento de Dados
Utilizamos serviços de nuvem seguros (Google Firebase) para armazenar seus dados. Retemos as informações coletadas pelo tempo necessário para fornecer o serviço solicitado. Quando armazenamos dados, protegemos dentro de meios comercialmente aceitáveis para evitar perdas e roubos, bem como acesso, divulgação, cópia, uso ou modificação não autorizados.

## 4. Compartilhamento de Informações
Não compartilhamos informações de identificação pessoal publicamente ou com terceiros, exceto quando exigido por lei.

## 5. Direitos do Usuário
Você é livre para recusar a nossa solicitação de informações pessoais, entendendo que talvez não possamos fornecer alguns dos serviços desejados. Você tem o direito de solicitar a exclusão dos seus dados a qualquer momento através dos nossos canais de suporte.

## 6. Compromisso do Usuário
O usuário se compromete a fazer uso adequado dos conteúdos e da informação que o Agenda Escolar IA oferece no site e com caráter enunciativo, mas não limitativo.
